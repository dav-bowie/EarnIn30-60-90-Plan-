JSON Prompt Methodology

My repeatable method for turning any idea into structured AI prompts

Write the Logline

Start with a clear 1-sentence concept that captures the core message

Define Constraints

Duration, platform, aspect ratio, brand tone - set boundaries upfront

Map Story Beats

Outline 3-5 story beats and map them to specific scenes

Define Shots

For each scene: environment, intent, and exact shots with durations

Audio Master

Add VO timestamps and music hit points for perfect sync

Global Controls

Seed, fps, negative prompts to keep outputs stable and consistent

Example JSON Structure

{
  "logline": "Morning café worker discovers instant cash access",
  "constraints": {
    "duration": "15s",
    "platform": "TikTok",
    "aspect_ratio": "9:16",
    "brand_tone": "Empowering"
  },
  "story_beats": [
    "Morning routine setup",
    "Financial stress moment", 
    "EarnIn discovery",
    "Instant relief",
    "Call to action"
  ],
  "scenes": [
    {
      "environment": "Café kitchen",
      "intent": "Establish routine",
      "shots": ["Wide establishing", "Close-up hands"],
      "duration": "3s"
    }
  ],
  "audio_master": {
    "vo_timestamps": [0, 5, 10],
    "music_hit_points": [2, 8, 14]
  },
  "global_controls": {
    "seed": 42,
    "fps": 24,
    "negative_prompts": ["blurry", "low quality"]
  }
}
